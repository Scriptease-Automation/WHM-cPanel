### Installation	

1. Download the installer script

```
curl -O 
https://raw.githubusercontent.com/Scriptease-Automation/whm-cpanel/main/whm_install.sh
```

2. Make it executable

```
chmod +x whm_install.sh
```

3. Run the installer

```
bash ./whm_install.sh -d <your domain> 
```
